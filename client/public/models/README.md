# 3D Models Folder

Place your 3D robot model here as `robot.glb`

## How to Get a Free Robot Model

### Option 1: Sketchfab (Recommended)
1. Go to: https://sketchfab.com/search?q=robot&type=models&features=downloadable
2. Search for: "humanoid robot" or "sci-fi robot"
3. Filter by: Downloadable (free)
4. Download as: **glTF** format (.glb)
5. Rename to: `robot.glb`
6. Place in this folder

### Recommended Models (Free):
- "Robot Kyle" - https://sketchfab.com/3d-models/robot-kyle-0f6b1d5b8e3e4b9e8f5c1d5b8e3e4b9e
- Search "robot android" or "robot assistant"

### Option 2: Mixamo
1. Go to: https://www.mixamo.com/
2. Sign in with Adobe account (free)
3. Choose a character
4. Download as FBX, then convert to GLB using https://gltf.report/

### Option 3: Ready Player Me
1. Go to: https://readyplayer.me/
2. Create a robot-style avatar
3. Download as GLB

## Model Requirements
- Format: `.glb` (preferred) or `.gltf`
- File size: Under 5MB for fast loading
- Filename: `robot.glb`

## After Placing the Model
The app will automatically load it from `/models/robot.glb`
